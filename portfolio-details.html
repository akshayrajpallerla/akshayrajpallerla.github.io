This project relates to a manufacturing company that produces cycling equipment and accessories. The data is in the form of raw CSV files and contains all the information about transactions, returns, products, customers and sales territories' main objective of the project is to build an executive dashboard and different reports so that the organization can track their different KPI measures on sales, revenue, profits, returns, product performance, customer analytics. This project allows the team to compare regional performance, analyze product-level trends, and identify high-value customers. The roadmap of this project is as follows : 1) Connecting raw data and data transformation. 2) Building a relational data model. 3) Create different measures and calculated columns using DAX. 4) Dashboard creation to visualize the data.
